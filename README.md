# G-CDSS

Genomics Clinical Decision Support System

## Project Overview

This project aims to build an explainable genomics-based clinical decision support system that integrates genomic variants, pharmacogenomics, clinical evidence, and structured patient information to generate evidence-based clinical recommendations.
